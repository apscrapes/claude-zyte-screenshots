# claude-zyte-screenshots
Capture a screenshot of any public URL using the Zyte API.

## Usage Example 
~~~
use zyte-screenshots skills and fetch screenshot of https://books.toscrape.com/catalogue/a-light-in-the-attic_1000/index.html , my Zyte API key is XXXXXXXX
~~~

## Claude Import

- Download the repo as .ZIP
- Open Claude and go to Settings > Capabilities
- Navigate to the Skills section
- Click Add Skill or Upload Skill
- Select your ZIP file (product-extractor-skill.zip)
- Claude will validate and load your skill
- Make sure "Domain allowlist" is set to "All domains"

- Enable the Skill
- After uploading, ensure the skill is enabled in Settings > Capabilities
- The skill will appear in your list of available skills
